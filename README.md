## Description

Simple nestjs starter project with authentication, simple role based authorization, and typeorm over a dockerized postgres instance.

## Installation

```bash
$ npm install
$ docker-compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
